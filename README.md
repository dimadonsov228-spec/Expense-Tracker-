# Expense-Tracker-
# Учёт расходов

Автор: Донцов Д.Д

Описание: Простое приложение для учёта расходов с возможностью добавления, фильтрации и подсчёта суммы за выбранный период.

Использование:
- Введите сумму, выберите категорию и дату, нажмите "Добавить расход".
- Используйте фильтры по категории и дате для просмотра и подсчёта расходов.

Пример теста:
- Добавьте расход "50" на "Еда" за "2023-10-01".
- Выберите категорию "Еда" и дату "2023-10-01" — увидите расход.
- Посчитайте сумму за период — получится 50.
